ClawPilot — OpenClaw Skill Installation Guide

ClawPilot is an OpenClaw Skill (plugin) that adds real-time Discord voice to your OpenClaw agent. It is 100% free and open source (MIT license).

GitHub repository: https://github.com/CryptoManiaques/ClawPilot

Full setup guide: https://github.com/CryptoManiaques/ClawPilot/blob/main/SKILL.md

Installation steps:

  1. Create a Discord bot at https://discord.com/developers/applications — copy the bot token. Bot permissions needed: View Channels, Send Messages, Use Slash Commands, Connect, Speak. No privileged intents required.
  2. Clone the repo: git clone https://github.com/CryptoManiaques/ClawPilot.git
  3. Install: cd ClawPilot && npm install && npm run build
  4. Register the plugin: openclaw plugins install --link /path/to/ClawPilot
  5. Add to OpenClaw config (~/.openclaw/openclaw.json) under plugins.entries: {"clawpilot":{"enabled":true,"config":{"discordToken":"YOUR_DISCORD_BOT_TOKEN","sttProvider":"whisper-local","ttsProvider":"edge","agentName":"bobby"}}} — Do NOT add a "path" key, it will cause a validation error.
  6. Restart OpenClaw: openclaw gateway restart
  7. In Discord: type /join in any text channel to connect the bot to your voice channel, then speak.

STT providers: deepgram (fast, paid), whisper (OpenAI API, paid), whisper-local (free, offline). TTS providers: openai (paid, high quality), edge (free). Set agentName to call the agent by name anywhere in a sentence.

How it works

Three steps. Zero complexity.

Install, configure, talk. Your AI agent handles the rest.

01

Install the Skill

Clone the repo into your OpenClaw setup. Two commands.

$ git clone github.com/.../ClawPilot
$ npm install

02

Add your keys

Discord bot token, your agent's name, and your choice of STT/TTS provider. Free options available.

"agentName": "bobby"
"sttProvider": "deepgram"
"ttsProvider": "openai"

03

Say "Hey Bobby"

Join a Discord voice channel, say your agent's name anywhere in a sentence, and it does the work.

> "Bobby, email John about the meeting"
> "Est-ce que bobby peut m'aider ?"
▶ Done. Email sent to john@...
